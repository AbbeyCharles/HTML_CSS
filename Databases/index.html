What are databases?

A structured set of data

Databases are a collection of information, they are used for creating, manipulating, and deleting data. There are many different types of databases and they can be classified in a basic approach of relational database or non-relational database. Relationships means a set of our data is related to another section of our data. Our databases are put into sections of data called schemas, inside each schema data set is organized into tables. Those tables contain columns and rows that house that particular set of data.
